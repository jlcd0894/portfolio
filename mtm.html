MTM

This was my first project as freelance, I was hired by a middleman which especializes in finding a designer that matches with it's client's design profile, it involved creating a logo that represented a new branch that specializes in a different field, while conserving elements from the parent company.

Wokring in this project helped me develop better communication skills because I had no personal interaction with the client, the only way I could get feedback was through the middleman so this made it a challenge because I had to convey a visual representation of what the client told him.

MTM is a branch of Mecatronic, a Tijuana based mechanical and industrial engineering company, this division is a metalworking company specialized in the creation, reparation and maintenance of plastic injection molds.

Tools used:

Challenge

Creating a brand identity for the new branch of a company, while maintaining visual queues that helps associating it to Mecatronic.

Strategy and Approach

The objective was utilizing colors, fonts and shapes similar to Mecatronic's logo to help reinforce the idea that even though it's a new branch they're still the same company.

Result

The end result used a similarity approach, the design used the same colors but changed the isotype now depicting a piece of machinery, while maintaining a similar structure from Mecatronic's logo to help make a correlation between the two brands.
